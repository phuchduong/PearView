# PearView 🍐

PearView is a whimsical yet powerful financial analytics suite from **Happy & Pear** designed to help investors find the "sweet spot" in market cycles. By transforming raw historical data into interactive visualizations, PearView provides clarity on momentum, value, and strategic accumulation windows.

## 🌟 Key Features

* **Interactive Price Analysis**: Dynamic charts built with ApexCharts for exploring Bitcoin and stock price history.
* **Moving Average Indicators**: Visual tracking of the 200-day moving average (DMA) to identify long-term trends.
* **Momentum Metrics**: Graphical representations of price velocity and midterm ROI.
* **Strategic Scoring**: Models designed to highlight recommended accumulation and buy periods.
* **Data-Driven Tables**: Searchable and sortable historical data powered by DataTables.

## 🛠️ Technical Stack

* **Frontend**: HTML5, CSS3 (Metronic Template), and JavaScript.
* **Visualization**: D3.js for custom SVG rendering and ApexCharts for interactive dashboards.
* **Data Processing**: Python for backend analytics and CSV parsing utilities.
* **UI Components**: DataTables for high-performance data grids.
